Prototipar (P4)

Modulo Principal

Bisturí 3.0

Ensamble final

Descripción:Modelo final del bisturi, con boton para realizar la accion "back" (enviar señal via bluetooth y ejecutar en Unity la accion Back).

Componentes internos (PCB)

Descripción:Componentes internos ensamblados, mediante el PCB mostrado en entregables anteriores.

Video: funcionamiento del modulo principal

Programación en Unity

Realidad Aumentada: Codificación en la interfaz Unity

Hide (Ocultar): Este código sirve para ocultar el objeto al momento de tener contacto con otro (bisturí).


                using System.Collections;
                using System.Collections.Generic;
                using UnityEngine;

                public class Hide : MonoBehaviour
                {
                    public GameObject TreeGroups;
                    void OnTriggerEnter(Collider other)
                    {
                        Renderer rend = TreeGroups.GetComponent();
                        if (other.tag == "Player")
                        {
                            rend.enabled = false;
                        }
                    }
                }

Show (Mostrar): Este código sirve para mostrar el objeto previamente ocultado al momento de presionar la tecla “W”.


                using System.Collections;
                using System.Collections.Generic;
                using UnityEngine;

                public class Show : MonoBehaviour
                {
                    public GameObject TreeGroups;
                    void Update()
                    {
                        Renderer rend = TreeGroups.GetComponent();
                        if (Input.GetKey(KeyCode.W))
                        {
                            rend.enabled = true;
                        }
                    }
                }

Función Hide and Back en Unity

Descripción: Presentación del funcionamiento de la función Hide and Back programada en Unity, la cual oculta secciones de la quemadura (representada como una matriz de cubos) a medida que el cilindro del video se posiciona sobre ellas. Posteriormente, al presionar la letra W, las secciones regresan a ser visibles.
