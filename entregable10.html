Codigo utilizado para enviar una señal blueetooth al presionar un boton

El código del programa permite que al pulsar un botón el módulo Bluettooth HC-05 mande una señal al dispositivo conectado a este. La señal es la letra “A” si es que en caso se tiene pulsado el botón; en cambio, si no se presiona este botón la señal es la letra “B”. Se hace este código con el fin de que al conectarse el módulo principal (Bisturí) con el celular que está en el módulo secundario (Visor), el programa que se encarga de la realidad aumentada reciba una señal diferente en caso se presione o no el botón ya que este botón tendrá la función de “Deshacer” una acción previamente hecha en la simulación.

                #define BUTTON 5 //el pin de entrada donde el pulsador esta conectado
                SoftwareSerial mipuerto =  SoftwareSerial ( rxPin , txPin ) ;

                mipuerto.begin(9600);

                pinMode(BUTTON,INPUT); //el boton es una entrada 
                }

                //Bucle

                val=digitalRead(BUTTON); //leer el valor de entrada y almacenarlo
