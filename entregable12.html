Programación en Unity

Codigo utilizado para enviar una señal blueetooth al presionar un boton

El siguiente código sirve para destruir un objeto en la simulación. Al poner este código en un objeto hace que este al entrar en contacto con otro cuerpo al cual le demos masa en sus características este se destruya y desaparezca de la simulación.


                using System.Collections;
                using System.Collections.Generic;
                using UnityEngine;

                public class destruirW : MonoBehaviour
                {

                    void OnTriggerEnter(Collider other)
                    {
                        if (other.tag == "Player")
                        {
                            Destroy(gameObject);
                        }
                    }
                }

El siguiente código al ser puesto en un objeto 3D permite mover éste dentro de la simulación mientras se esté presionando con el clic derecho del mouse. Es necesario incluir funciones como "Event Systems" para que funcione de manera correcta.


                using UnityEngine;
                using UnityEngine.EventSystems;
                public class DragScene : MonoBehaviour, IDragHandler
                {
                    public float z = 0.0f;
                    public void OnDrag(PointerEventData eventData)
                    {
                        Vector3 mousePosition = Input.mousePosition;
                        mousePosition.z = z;

                        transform.position = Camera.main.ScreenToWorldPoint(mousePosition);
                    }
                }
